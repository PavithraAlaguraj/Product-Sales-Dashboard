# Data Analytics Projects

## 1. Product Sales Dashboard

### Project Overview
Developed an interactive Power BI dashboard to analyze sales performance, revenue trends, and regional sales distribution to support business decision-making.

### Tools Used
- Power BI
- Power Query
- Excel

### Key Features
- Performed data cleaning and transformation using Power Query.
- Created interactive dashboards to analyze sales trends and revenue performance.
- Developed visualizations including bar charts, pie charts, KPI cards, and tables.
- Implemented slicers and filters for dynamic data exploration.
- Analyzed regional sales performance and business trends.
- Generated actionable insights to support data-driven decision-making.

## 2. Enterprise Sales Performance Analytics Dashboard

### Project Overview
Designed and developed an enterprise-level sales analytics dashboard to monitor business performance, identify trends, and support strategic decision-making.

### Tools Used
- Power BI
- Power Query
- Excel

### Key Features
- Cleaned and transformed large sales datasets for analysis.
- Developed data models and established relationships between multiple tables.
- Created KPI cards, bar charts, line charts, and interactive reports.
- Implemented DAX measures to calculate key business metrics.
- Analyzed sales trends, customer behavior, and business performance patterns.
- Built interactive filters and slicers for enhanced user experience.
- Delivered actionable insights to support strategic business planning.
